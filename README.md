# vue-model-3.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
### Upload the built code to the server
```
npm run upload
```
### Publishing code to the server
```
npm run deploy
```

### Eslint-Rules
See [.eslintrc.js].
