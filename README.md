# vue-model-3.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Eslint-Rules
See [.eslintrc.js].

### Directory Structure
src
  api         业务api请求封装（尽量不处理数据处理）
  assets      项目资源目录
  components  组件目录
  lib         基础依赖库
  router      路由模块
  store       状态管理
  views       页面资源目录
  main.js     入口文件
  App.vue
package.json  依赖包版本库
vue.config.js 项目规则配置
